# Streamlining Access to RCSB PDB APIs with Python: Sequence and Models
This workshop focused on utilizing the `rcsb-api` package for accessing the RCSB PDB Sequence and Model APIs. Additional documentation can be found on [readthedocs](https://rcsbapi.readthedocs.io/en/latest/index.html).

## Webinar | May 19, 2026
Jupyter notebooks used for the RCSB PDB API Webinar, "Streamlining Access to RCSB PDB APIs with Python: Sequence and Models". Additional Jupyter notebook tutorials can also be found in the [py-rcsb-api repository](https://github.com/rcsb/py-rcsb-api/tree/master/notebooks)
